# jabr

**jabr** — это безопасная и удобная обёртка на Rust для [Java Access Bridge](https://docs.oracle.com/javase/accessbridge/2.0.2/index.html).

Java Access Bridge API позволяет разрабатывать вспомогательные технологические приложения для операционной системы Microsoft Windows, работающие с Java-приложениями. Он содержит нативные методы, которые позволяют просматривать и управлять информацией о графических элементах пользовательского интерфейса (GUI) в Java-приложениях. Эта информация передаётся через Java Access Bridge в ваши вспомогательные технологические приложения.
