## JAB API

Java Access Bridge API позволяет вам разрабатывать приложения вспомогательных технологий для операционной системы Microsoft Windows, использующих Java-приложения. Он содержит нативные методы, которые позволяют вам просматривать и управлять информацией о GUI-элементах в Java-приложениях, передаваемой через Java Access Bridge в ваше приложение вспомогательных технологий.
[Оригинальная документация интерфейса](https://docs.oracle.com/javase/9/access/jaapi.htm#JSACC-GUID-C10D11B0-F588-43FA-BBDE-70E9085E9AFF)
